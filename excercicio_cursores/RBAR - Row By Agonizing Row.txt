O conceito de abordar dicotomia entre o processamento de dados de forma a sequência, através do uso de cursores e do processamento de dados por uma abordagem de conjunto. Normalmente em procedures armazenadas, se faz muito uso de cursores para processamento de dados, porém, para casos em que esse processamento vai ser escalado para um o volume de dados muito maior, é uma estratégia mais eficiente codificar usando processamento de dados em uma única vez.
